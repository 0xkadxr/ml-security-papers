## Rating System

Each paper is rated on two dimensions:

| Dimension | Description |
|-----------|-------------|
| **Relevance** | How central the paper is to ML security and red-teaming (1-5 stars) |
| **Impact** | Influence on the field, citation count, and practical significance (1-5 stars) |

## How to Use

1. **Browse by category** -- Click on any category in the Table of Contents to explore papers in that area.
2. **Start with high-impact papers** -- Look for papers rated 5 stars in both Relevance and Impact.
3. **Follow the reading order** -- Within each category, papers are roughly ordered from foundational to recent.
4. **Check the resources** -- Visit [conferences](resources/conferences.md) and [blogs](resources/blogs.md) for additional learning materials.

### Suggested Reading Paths

- **New to ML Security?** Start with [Adversarial ML](papers/adversarial-ml/) foundational papers, then move to [Prompt Injection](papers/prompt-injection/).
- **LLM Red-Teaming Focus?** Start with [Jailbreaking](papers/jailbreaking/), then [Prompt Injection](papers/prompt-injection/), then [Agent Security](papers/agent-security/).
- **Multimodal Systems?** Read [Adversarial ML](papers/adversarial-ml/) basics, then [Multimodal Attacks](papers/multimodal-attacks/).
